# FizzBuzz

A simple JavaScript implementation of the classic **FizzBuzz** programming exercise.  
This project demonstrates core programming fundamentals including:

- Loops  
- Conditional logic  
- Branching  
- Modulo operations  
- Clean, readable code structure  

FizzBuzz is often used as an introductory coding challenge to verify understanding of control flow and logical evaluation.

---

## 🚀 How It Works

The program prints numbers from **1 to 100**, applying the following rules:

- If a number is divisible by **3**, print `"Fizz"`
- If divisible by **5**, print `"Buzz"`
- If divisible by **both 3 and 5"**, print `"FizzBuzz"`
- Otherwise, print the number itself
